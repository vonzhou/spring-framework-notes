## Eclipse shortcut

Ctrl + T / F4  : 类型层次结构

Ctrl + O : Outline

Ctrl+M ： 最大化窗口

Ctrl + Shift + F : 格式化代码

Ctrl + Shift + R : 全局搜索，类似intellij的 double shift
